# Tweet IDs for: Public Discourse on Low Fertility in Japan

## Overview

This repository provides the Tweet IDs used in the study:

**"Public Discourse on Low Fertility in Japan: A Comparative Analysis of Parliamentary Debate, News Media, and Social Media."**

The dataset contains Tweet IDs collected from Twitter (now X) using the Academic API between **1 January 2014 and 31 May 2023**. Tweets were retrieved if they contained the Japanese keywords:

* 少子化 (shōshika; low fertility)
* 出生率 (shusshō-ritsu; fertility rate)

In total, the Twitter corpus contains **5,614,201 tweets**.

In accordance with Twitter/X's Developer Policy, only Tweet IDs are shared.

---

## Data Collection

Tweets were collected via the **Twitter Academic API**.
The query retrieved posts containing at least one of the following terms:

* 少子化
* 出生率

Time range:

```
2014-01-01 to 2023-05-31
```

---

## Dataset Format

The file contains **one Tweet ID per line**, sorted in ascending order.

Example:

```
123456789012345678
123456789012345679
123456789012345680
```
